### 안녕하세요, **운영과 개발을 모두 경험한** 이태형입니다.

(주)나임네트웍스에서 Angular 기반 프론트엔드 개발을 담당하며 개발 프로세스와 협업 능력을 키웠습니다.

이후 서비스의 전체 생명주기를 이해하고자 인프라 영역으로 확장하여, 현재 (재)건설산업교육원에서 **LMS 플랫폼의 시스템 운영, 간단한 기능 개발, 외주 업체 관리**를 담당하고 있습니다. 또한 반복 문의를 줄이기 위해 **클로드 AI 도구를 활용하여 FAQ 챗봇을 직접 기획하고 개발**했습니다.

개발 경험을 바탕으로 시스템을 이해하고, 운영 경험을 통해 안정적인 서비스를 제공할 수 있는 **IT 엔지니어**로 성장하고 있습니다.
