### 안녕하세요, **기획·인프라·개발을 경험한** 이태형입니다.

(주)나임네트웍스에서 클라우드 통합 솔루션 '탱고'의 Angular5 기반 프론트엔드 개발 및 유지보수를 담당하며, 디자이너 및 백엔드 개발자와의 협업을 통해 UI/UX 개선과 원활한 커뮤니케이션 능력을 키웠습니다.

이후 서비스의 전체 생명주기를 이해하고자 인프라 영역으로 관심을 확장하여, 현재 (재)건설산업교육원에서 LMS 플랫폼 인프라를 운영하며 서버 관리, DB 운영, 보안 솔루션 운영 및 Python 기반 업무 자동화를 담당하고 있습니다.

프론트엔드 개발 경험과 인프라 운영 역량을 결합하여, 사용자 경험부터 시스템 안정성까지 고려할 수 있는 Full-cycle 개발자로 성장하는 것이 목표입니다.
